# node_design_pattern

Description: This is a project to understand design pattern in Nodejs, such like signleton, object, module, export ...

## Installation

```
npm install

```

## Usage

** Singleton Example
* signleton object 1
* */singleton/1/:id*

* signleton object 2
* */singleton/2/:id*

** Object Example
* object 1
* */object/1*
* object 2
* */object/2*

** few example about creating modules, functions and objects
* */df/df*
* */df/anony*
* */df/named*
* */df/obj/1*
* */df/obj/2*
